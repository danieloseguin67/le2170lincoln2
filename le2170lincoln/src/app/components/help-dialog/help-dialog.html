help_outline How Listings Management Works

Use this screen to manage all apartment listings for 2170 Lincoln.

  • Add Listing: opens a form to create a new apartment with ID, titles (FR/EN), type, size, rent, availability, bedrooms, bathrooms and features.
  • Edit: opens the same form pre-filled so you can update an existing listing.
  • Remove: permanently removes the listing from the table.
  • Save: saves the current list of apartments (for production this would write to the SQL Server database).

The table shows both French and English titles so you can verify bilingual data side by side.
