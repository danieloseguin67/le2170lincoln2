# Background Image Instructions

To add your Montreal skyline background image:

1. Replace this file with your actual Montreal skyline image
2. Name the file: montreal-skyline.jpg (or update the CSS path if using a different name/format)
3. Recommended image specifications:
   - Format: JPG or WebP for best performance
   - Resolution: At least 1920x1080 pixels (HD) for sharp display on all devices
   - Aspect ratio: Wide landscape (16:9 or similar)
   - File size: Optimize for web (ideally under 500KB)

## Supported formats:
- montreal-skyline.jpg
- montreal-skyline.jpeg
- montreal-skyline.webp
- montreal-skyline.png

The CSS is currently configured to use: montreal-skyline.jpg

If you want to use a different filename, update this line in home.scss:
url('/assets/images/montreal-skyline.jpg')
